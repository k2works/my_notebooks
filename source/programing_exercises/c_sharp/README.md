# FizzBuzz

## 仕様
+ 3で割り切れる場合は「Fizz」を出力する
+ 5で割り切れる場合は「Buzz」を出力する
+ 両方で割り切れる場合は「FizzBuzz」を出力する
+ 上記以外の場合は数字を返す
+ 繰り返し実行できるようにする
+ タイプごとに出力を変える
